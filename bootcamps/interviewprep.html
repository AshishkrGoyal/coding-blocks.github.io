Operating Systems

In this bootcamp you will learn all important Operating System concepts like Memory Management, Virtual Memory, Deadlocks, Multi-threading, Process Synchronisation and UNIX Commands.

DBMS

A database management system (DBMS) is a computer software application that interacts with the user, other applications, and the database itself to capture and analyze data. In this bootcamp we will look at schema design, SQL Queries, and Normalisation Techniques.

Networking

Students will also learn about Networking Protocols, Routing algorithms in much more detail .

System Design

A comprehensive analysis of system design problems like URL Shortner, Google Ranking algorithms, Elevator Design, Facebook Ranking system etc will be covered.
